How to Use:
1. Copy all the folders into xampp/htdocs/Laravel (wherever your laravel folder is)
2. Enable GD in php.ini in xampp/php/ by removing the ";" on the line ;extension=gd if it isn't removed already.
3. Navigate to xampp/htdocs/Laravel
4. Run server with php artisan serve on the terminal
5. Link the storage with php artisan storage:link to ensure the images get saved correctly.
6. Access localhost:8000, upload a file and convert with the available options.
